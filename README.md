# 🎓 School LC System

A simple and smart School Leaving Certificate (LC) Generator web application.

## 🚀 Features
- Generate School Leaving Certificate
- Clean and simple UI
- Fast and easy to use
- Printable LC format

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📂 Project Files
- index.html – Main page
- style.css – Design
- app.js – Functionality

## 📸 Output
Generate LC easily with student details.
